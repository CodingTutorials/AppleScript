# Display Dialog

Write display dialog in script editor.

```
display dialog
```

Then add a space and 2 quotaion marks with text in between

```
display dialog "hello world"
```

Click "run"

## Examples

### Example 1:
```
display dialog "hi there!"
```

### Example 2:
```
display dialog "hey yo sup"
```

### Example 3:
```
display dialog "abc123"
```
